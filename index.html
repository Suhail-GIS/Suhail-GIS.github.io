Watershed Boundary Mapping (California, USA)

This is my first GIS project where I mapped the Navarro watershed boundary and generated streamlines using ArcGIS Pro. It helped me understand basic hydrological mapping and topographic referencing.

Village Mapping Using QGIS (Khajurahat, India)

This project focuses on mapping my native village, Khajurahat in Ayodhya, using QGIS. It includes spatial layers such as roads, water bodies, land use, and infrastructure, along with attribute tables to analyze socio-economic data.
